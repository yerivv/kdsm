반품 신청 절차
  1. 반품 가능 내역 조회
  2. 목록 선택 후 반품신청서 작성
  3. 수량 및 사유 입력
  4. 입점업체 담당자 확인 및 승인처리
  5. 반품 수거 (업체별 택배 상이)
  6. 반품 최종 확인 및 환급 완료
꼭 참고해주세요!
  • 반품된 제품은 유효기간과 관계없이 전량 폐기처분 되오니 신중하게 신청바랍니다.
  • 반품 신청 시 담당 영업사원이 방문하여 확인 후 접수 진행됩니다.
  • 반품 접수 완료 건에 한해 해당 월안에 마이너스 잔고로 반영되며 최종금액은 마이페이지 내 거래원장에서 확인가능합니다.
  • 보여드리는 예상 반품금액에는 입력하신 낱알반품 예상금액도 반영되어 확인됩니다. 하지만 특정 상품일 경우 낱알 개수 산정이 안될 수 있으니 이점 참고부탁드립니다.
  • 현재 판매되고 있지 않은 상품인 경우, 조회가 안될 수 있으니 담당영업사원에게 요청하시면 조치 가능하십니다.
  • 반품신청 내역과 실물이 일치하도록 해주세요. 영업사원 방문 시 수량 / 제품 검수 진행하여 신청내역과 다를 시 수정될 수 있습니다.
  • 낱알 수량 입력방법 : 예) ooo상품 10캡슐*12개 -> 낱알수량 119개까지 입력 가능
반품신청 가능한 조건
  • 상품별 반품 기간은 상이 할 수 있으며, 반품 기한이 경과된 경우 반품이 불가 할 수 있습니다.
  • 의약품 : 구매일로부터 6개월 이내의 재판매가 가능한 상품
  • 의약외품 및 건강기능식품 : 구매일로부터 6개월 이내의 재판매가 가능한 상품
  • 동물약/용품 : 구매일로부터 6개월 이내의 재판매가 가능한 상품
  • 약국전용 화장품 : 구매일로부터 3~6개월 이내의 재판매가 가능한 상품
반품처리 불가 조건
  • 회원에게 책임있는 사유로 상품이 멸실/훼손된 경우(반품포장 부실, 가격택 또는 스티커 부착, 마킹 등)
  • 회원의 사용 또는 일부 소비에 의해 상품의 가치가 현저히 감소한 경우
  • 시간의 경과에 의하여 재판매가 곤란한 정도로 상품 등의 가치가 현저히 감소한 경우
  • 판매자가 판매 시 별도로 반품 가능 상품이 아니라는 사실을 고지한 경우
  • 타임세일 및 공동구매를 통한 주문
    • 반품쿠폰 사용조건 : 반품 기한 내의 상품만 반품쿠폰 사용가능하며, 반품쿠폰 1장당 1개 업체에만 쿠폰 적용이 가능합니다.
    • 주문상품 수령 후, 반품 절차 진행 시 결제수단과 무관하게 오직 "예치금"으로만 환불이 가능합니다.
    • 의약품 반품은 약가인하/약가 인상 보상이 불가하며, 낱알 반품의 보상 기준가는 반품확정 시점의 복지부 고시가 기준으로 합니다.
    • 반품완료 처리 일자가 국세청 폐업 신고 이후 일 경우 세금계산서 발행이 불가합니다. (세금계산서 선발행을 원하실 경우, 고객센터로 연락 부탁드립니다.)
